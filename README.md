#  Recursos Humanos

Aplicación de escritorio desarrollada en **Java** utilizando el patrón **DAO (Data Access Object)** y **Programación Orientada a Objetos (POO)**.  
Su propósito es permitir la **gestión de funcionarios** de una organización, implementando las operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

---

##  Objetivo

Diseñar y construir una aplicación que permita listar, crear, editar y eliminar información correspondiente a funcionarios, integrando el patrón DAO y el manejo de excepciones en Java.

---

##  Tecnologías utilizadas

- **Lenguaje:** Java 17  
- **Framework:** Maven  
- **Interfaz:** Java Swing  
- **Patrón de diseño:** DAO  
- **Motor de base de datos:** MySQL (puede adaptarse a MariaDB o PostgreSQL)  
- **IDE recomendado:** IntelliJ IDEA o NetBeans
